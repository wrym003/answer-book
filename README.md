# 答案之书（全程由codex编写）

一个简单的网页小工具。打开页面后，心里想着一个问题，点击按钮，就会随机出现一句回答。

## 在线访问

访问地址：

https://wrym003.github.io/answer-book/

## 功能

- 随机显示一句答案
- 适配电脑和手机屏幕
- 不需要登录，不需要安装软件
- 使用 HTML、CSS 和 JavaScript 制作

## 文件说明

- `index.html`：网页结构
- `styles.css`：页面样式
- `script.js`：随机答案逻辑

## 如何修改答案

打开 `script.js`，找到 `answers` 列表，在里面增加、删除或修改文字即可。

例如：

```js
"现在就开始。"
```

每一句答案都需要放在英文双引号里，句子之间用英文逗号分隔。

## 项目状态

这是一个适合初学者练习的静态网页项目，可以继续增加更多答案、动画效果、背景音乐或分享按钮。
